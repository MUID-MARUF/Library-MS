# Library Management System - Project Progress

## 📋 Project Overview
A web-based Library Management System using a provided MySQL schema as the foundation.

## 🛠 Tech Stack
- **Frontend:** HTML5, Vanilla CSS, JavaScript (ES6+)
- **Backend:** Python Django
- **Database:** MySQL (via XAMPP)
- **UI Style:** Simple, elegant, and modern

## 🏗 Database Schema (The Root)
The system is built upon the existing `Library Management System.txt` which includes:
- **Core Tables:** Author, Publisher, Member, Staff, Category, Book, Issue, Rating.
- **Relations:** Writes, Publish, Contain, Handles, HasBook, Generate.
- **Triggers:** Data validation on inserts.
- **Stored Procedures & Views:** Provided for data access.
